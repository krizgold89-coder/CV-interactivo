# CV Interactivo - Cristofer Henríquez Henríquez

Proyecto de Currículum Vitae interactivo desarrollado como parte de la Evaluación Final del Módulo 2

## Demo

- ver aqui: (git@github.com:krizgold89-coder/CV-interactivo.git)

## Tecnologías Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
- Git & GitHub

## Funcionalidades

1. Diseño responsivo con Bootstrap.
2. Navegación con smooth scroll entre secciones.
3. Modo oscuro/claro activado desde la barra de navegación.
4. Validación de formulario de contacto en tiempo real.
5. Animaciones hover en tarjetas de experiencia.
6. Navbar colapsable en dispositivos móviles.
